# content-seo — grow reach of the writing

Weekly loop over the site's content (`src/content/posts`, `src/content/til`). Consumes
PostHog analytics (`.env` `POSTHOG_API_KEY`), search/SERP signals, and a topic backlog;
produces post/TIL drafts, SEO-fix PRs (via `/pr`, gated by `pnpm verify`), and topic
`signals`. It points at artifacts in `signals/`/`docs/`; it never holds them.

## Current focus
Fix on-page SEO hygiene surfaced by the first audit — meta-description length is the
first lever (11/13 posts truncate in SERPs). See [[meta-descriptions-too-long]].
