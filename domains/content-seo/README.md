# content-seo — grow reach of the writing

Weekly loop over the site's content (`src/content/posts`, `src/content/til`). Data sources,
in priority order: **PostHog** (on-site behavior — see [[analytics-events]]) and **Google
Search Console** (your real queries/impressions/rankings — free; not yet connected). Paid
SERP scrapers (Serper / DataForSEO) are *optional, later* — only for competitor and
keyword-universe research beyond your own data. Produces post/TIL drafts, SEO-fix PRs (via
`/pr`, gated by `pnpm verify`), and topic `signals`. Links artifacts; never holds them.

## Current focus
Fix on-page SEO hygiene surfaced by the first audit — meta-description length is the
first lever (11/13 posts truncate in SERPs). See [[meta-descriptions-too-long]].
